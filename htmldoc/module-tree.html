Module Hierarchy

  • tnef: The namespace package imports the higher-level API to TNEF decoding
    • tnef.cmd: This module implements a low-level wrapper to the 'tnef' command.
    • tnef.config: This module contains only configuration settings.
    • tnef.errors: This module contains exception & error definitions
    • tnef.rtfparser: The module implements a pyparsing - based special-purpose RTF parser specifically for extracting HTML from RTF contained within a TNEF attachment.
    • tnef.tnef: The tnef module provides high-level access to tnef decoding; namely, listing contents of TNEF attachments and extracting and retrieving TNEF body/bodies and embedded files.
    • tnef.util: A few utility functions used by other modules.
