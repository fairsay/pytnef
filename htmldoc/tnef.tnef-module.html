Module tnef

source code

The tnef module provides high-level access to tnef decoding; namely, listing contents of TNEF attachments and extracting and retrieving TNEF body/bodies and embedded files.

Author: Petri Savolainen

getBody(sourcefile, preference=None, extract=True)

source code 

By default, looks up body types in default order (rtf > html > txt).

Parameters:
  • preference (tuple) - The lookup preference can be altered by giving a tuple consisting one or more out of "rtf", "html" and "txt".
  • extract - return html extracted from inside rtf instead of returning rtf? @rtype string

    If no body is found, an exception is raised.

  • sourcefile (open file (in mode 'rb'))
